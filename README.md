Integrative Project Summary (PI)

The Integrative Project (PI) aimed to develop a technological system focused on solving a real problem. The methodology used in this project included requirement gathering, planning, system development, and testing.

During the development process, technologies such as HTML, CSS, JavaScript, and database integration were used to ensure functionality and a good user experience. The project was designed to be efficient, practical, and aligned with user needs.

The results demonstrated that the proposed solution is viable and effective, meeting the defined requirements and improving the analyzed process. The system provides a clear example of how technology can be applied to solve real-world challenges.
